Crystal Builder

Simple program to build crystals for DFT/MD simulations

Home Documentation Install Release info View on GitHub

Description:

This is an utlity designed to construct a wide range of crystal structures for use in several common Density Functional Theory and Molecular Dynamics packages.

Capabilities:

  • Bravais lattices: Cubic, Orthorhombic, Tetragonal

  • Library structures:

    • 2D materials: Graphene, h-BN, (M)etal (D)i(C)halcogenide-2H/1T (i.e. SnS2), (T)ransition (M)etal (D)i(C)halcogenide-2H/1T (i.e. MoSe2)

    • Energetic materials (PETN-I, TATB, Î²-HMX)

  • examples/ Directory containing several codes demonstrating the usage of various library methods

  • Makefile.template Template to be customized for the users' environment

Usage:

See the directory examples/libxxx for some examples of calls to the various library subroutines. This directory also has a Makefile template for use in building your own codes against the library.

Alternatively, you can build your programs with the libraries as follows,

 icpc -L$(LIB_DIR)/libs -llammps -lmathns ... -I$(INC_DIR)/include code.cpp -o out.x
