Crystal Builder

Simple program to build crystals for DFT/MD simulations

Build options

This application is designed to be driven by a GUI with the final crystal structure being rendered in 3D and displayed on the screen. However, there a 4 ways in which the application can be built:

  1. GUI + 3D graphics rendering

  2. GUI only, no 3D graphics rendering

  3. Command line interface + 3D graphics rendering

  4. Command line interface only, no 3D graphics rendering

For option 1, the GTK+ 2.0 and Open GL libraries must be installed and in the usual path. If they are not, see the section below on Dependencies.

For option 2, only the GTK+ 2.0 libraries are required and in the usual path. If they are not, see the section below on GTK dependencies. Additionally, you must comment out the line defining the Open GL libraries in the Makefile, see below

## Mac OS X uses "frameworks"
ifeq ($(OSTYPE), Darwin)
GL = -framework GLUT -framework OpenGL -DOGL
endif
## Linux uses "libraries"
ifeq ($(OSTYPE), Linux)
GL = -lGL -lGLU -lglut -DOGL
endif

For option 3, only the Open GL libraries are required and in the usual path. If they are not, see the section below on GL dependencies. Additionally, you must comment out the line defining the GTK libraries in the Makefile, see below

## Load the GTK.  Use pkg-config to define the proper
## CFLAGS and LD_LIB parameters
GTK=`pkg-config --cflags --libs gtk+-2.0` -DUSE_GTK

For option 4, none of the external libraries are required. Additionally, you must comment out the line defining the GTK and Open GL libraries in the Makefile, see below

## Mac OS X uses "frameworks"
ifeq ($(OSTYPE), Darwin)
GL = -framework GLUT -framework OpenGL -DOGL
endif
## Linux uses "libraries"
ifeq ($(OSTYPE), Linux)
GL = -lGL -lGLU -lglut -DOGL
endif


## Load the GTK.  Use pkg-config to define the proper
## CFLAGS and LD_LIB parameters
GTK=`pkg-config --cflags --libs gtk+-2.0` -DUSE_GTK

Dependencies

GTK+ 2.0

  1. Linux: Fedora 21/RHEL 6.X/CentOS 6.X 
    yum install gtk+-devel gtk2-devel
  2. Linux: Debian/Ubuntu 
    apt-get install libgtk2*
  3. Mac OS X 10.9.X: Install homebrew if needed and then run the following 
    brew install gtk2
  4. Windows: Not supported

Open GL

  1. Linux: Fedora 21/RHEL 6.X/CentOS 6.X 
    yum install freeglut-devel mesa*
  2. Linux: Debian/Ubuntu 
    apt-get install freeglut3-dev
  3. Mac OS X 10.9.X: Mac OS X comes with Open GL / GLUT installed by default
  4. Windows: Not supported

Install

NOTE : Several options are listed on line 16 of the Makefile controling the environment and dialog verbosity. Specifically, the rendering window can be built with a default white background or black. Additionally, there are several macros for controlling how much information is printed to the terminal during a build session.

Once finished, simply run the following command in the top-level directory to build the application 

make all
